# ⏱️ Stopwatch Web App

A clean and responsive stopwatch web application built using **HTML, CSS, and JavaScript**.  
It allows users to start, pause, and reset time easily.

---

## 🚀 Features
- Start / Pause stopwatch
- Reset timer
- Millisecond precision
- Simple and responsive UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Click Start to run the stopwatch
